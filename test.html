Detta är en tutorial för HTML

Här får du lära dig hur man använder HTML och CSS.

Som standard skriver man <!DOCTYPE html>

Efter det ska du ha en tagg som heter <head>, där man specificerar information om dokumentet tex så lägger man länkar till CSS och JavaScript här.

En <body> måste även finna för det är detta dom är själva webbsidan

För att kunna skriva ut <p> som ska synas på webbsidan utan att den ska tolkas så får man skriva ett &-tecken och L+T för larger than.

Eller &-tecken, G+T som står för Greater Than.

Här kommer exempel på olika taggar. Klicka på en för att se vad dom gör.
